# Wedding Story Front End

React front end with local dev powered by vite

Serves as the front end product for developing a wedding story for a couple
Final product should be able to take in images to form stylized characters in couples likeness (using openai or image model)
and should later take in a questionnaire/story to create chapters from couple's relationship to form into a static site to be used in 
conjuction with wedding invites.

Requires the Wedding Story API to function


Run 
`npm run dev`
